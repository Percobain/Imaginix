# Imaginix

Imaginix is a modern web application built with Next.js, React, and various other technologies. This project aims to provide a seamless user experience with a robust and scalable architecture.

## Tech Stack

- **Next.js**
- **@fal-ai/serverless-client**
- **@excalidraw/excalidraw**

## Getting Started

Follow these instructions to set up the project locally.

### Prerequisites

Ensure you have the following installed:

- Node.js (>= 18.0.0)
- npm or yarn

### Installation

1. **Clone the repository:**

    ```sh
    git clone https://github.com/your-username/imaginix.git
    cd imaginix
    ```

2. **Install dependencies:**

    Using npm:
    ```sh
    npm install
    ```

    Using yarn:
    ```sh
    yarn install
    ```

### Running the Application

1. **Start the development server:**

    Using npm:
    ```sh
    npm run dev
    ```

    Using yarn:
    ```sh
    yarn dev
    ```

    The application will be available at `http://localhost:3000`.

2. **Build the application for production:**

    Using npm:
    ```sh
    npm run build
    ```

    Using yarn:
    ```sh
    yarn build
    ```

3. **Start the production server:**

    Using npm:
    ```sh
    npm start
    ```

    Using yarn:
    ```sh
    yarn start
    ```
